## 🧮 이 견적 계산표, 초등학생도 이해하는 설명

### 🎯 전체 개념: "이 책을 만들면 얼마야?"를 자동으로 계산하는 표예요

---

## 📋 1단계: 책의 기본 정보를 입력해요 (B열 - 노란색 칸)

| 항목 | 입력값 | 의미 |
|------|--------|------|
| **판형** (B3) | `B5` | 책의 크기 이름 |
| **페이지** (B4) | `2001` | 총 몇 페이지? |
| **부수** (B5) | `10` | 몇 권 만들어요? |
| **인쇄** (B6) | `IX-Eco` | 어떤 인쇄 방식? |
| **내지종이** (B7) | `모조80` | 안쪽 종이 종류 |
| **내지양/단면** (B8) | `양면` | 앞뒤 다 인쇄? |
| **표지종이** (B10) | `아트지250` | 표지 종이 종류 |
| **코팅** (B12) | `유광코팅` | 표지 코팅 방식 |
| **제본** (B13) | `중철` | 책 묶는 방식 |

---

## 🔍 2단계: 숨겨진 규격표에서 정보를 자동으로 찾아요 (H2:I4)

> 💡 오른쪽에 보이지 않는 "규격표"가 있어요. B3에 책 크기를 입력하면, 거기서 자동으로 찾아줘요!

**내지 규격 (H3):**
```
=VLOOKUP(B3, G6:J10, 2, FALSE)
```
> 🗣️ "B3(=B5)라는 판형을 G6:J10 표에서 찾아서, 그 옆의 두 번째 칸(규격)을 가져와!"
> → 결과: **46판**

**표지 규격 (H4):**
```
=INDEX(H13:M16, MATCH(B3,G13:G16,0), MATCH(B13,H12:M12,0))
```
> 🗣️ "B3(판형=B5)가 G열에서 몇 번째 줄인지 찾고, B13(제본=중철)이 H12:M12에서 몇 번째 칸인지 찾아서, 그 교차점의 값을 가져와!"
> → 결과: **국4절**

---

## 💰 3단계: 각 항목별 단가를 자동으로 찾아와요 (C열)

### 🖨️ 인쇄단가 (C6)
```
=INDEX(P4:Y16, MATCH(P2,O4:O16,1), MATCH(B6,P3:Y3,0))
```
> 🗣️ "인쇄 수량에 맞는 행, 인쇄 방식(IX-Eco)에 맞는 열을 찾아서 단가 표에서 값을 꺼내와!"
> → **25원/장**

### 📄 내지 양/단면 단가 (C8)
```
=VLOOKUP(B8, G21:H22, 2, FALSE)
```
> 🗣️ "B8(양면)을 작은 표에서 찾아서 단가를 가져와!"
> → **4.4625원**

### 🧾 면지 단가 (C9)
```
=INDEX(AI52:AM60, MATCH(B9,AH52:AH60,0), MATCH(B3,AI51:AM51,0))
```
> 🗣️ "면지 종류(없음)랑 판형(B5)로 표에서 찾아!"
> → **0원** (없음이니까!)

### 📑 표지 종이 단가 (C10)
```
=INDEX(AB52:AE63, MATCH(B10,AA52:AA63,0), MATCH(H4,AB51:AE51,0))
```
> 🗣️ "표지 종이(아트지250)와 표지 규격(국4절)으로 단가 찾아!"
> → **77.735원**

### 🎨 코팅 단가 (C12)
```
=INDEX(P19:U22, MATCH(B3,O19:O22,0), MATCH(V19,P18:U18,0))
```
> 🗣️ "판형(B5)과 코팅 종류로 단가 찾아!"
> → **100원**

### 📚 제본 단가 (C13)
```
=INDEX(P30:T49, MATCH(B5,O30:O49,1), MATCH(B13,P29:T29,0))
```
> 🗣️ "부수(10권)와 제본 방식(중철)으로 단가 찾아!"
> → **1,000원**

---

## 🧾 4단계: 최종 1부 단가를 계산해요 (D열)

| 항목 | 계산식 | 의미 |
|------|--------|------|
| **인쇄** (D6) | `=B4 × C6` = 2001 × 25 | 페이지 수 × 장당 인쇄비 |
| **내지** (D8) | `=C8 × B4` = 4.46 × 2001 | 내지 단가 × 페이지 수 |
| **면지** (D9) | `=C9` | 단가 그대로 |
| **표지** (D10) | `=C10` | 단가 그대로 |
| **코팅** (D12) | `=C12` | 단가 그대로 |
| **제본** (D13) | `=C13` | 단가 그대로 |

## 🎁 스프링 제본 특별 처리 (B14, C14, D14)

```
=IF(ISNUMBER(SEARCH("스프링", B13)), "타공", "")
```
> 🗣️ "만약 제본 방식에 '스프링'이 들어있으면 '타공'이라고 표시해, 아니면 빈칸!"
